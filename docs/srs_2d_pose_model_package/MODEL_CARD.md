# Model Card

## 名称

SRS 2D Pose Skeleton Extractor

## 任务

从棒球击球/投球视频中提取稳定的 2D 人体骨架、逐帧关键点、基础关节角度和帧质量标签。

## 模型组成

- 底层人体姿态估计：MediaPipe Pose，`model_complexity=2`
- 视频模式：`static_image_mode=False` + `smooth_landmarks=True`
- 融合策略：静态逐帧检测补充视频追踪中丢失或不可靠的快速肢体点
- 稳定策略：坏帧剔除、缺失关键点插值、核心/四肢/快速点分层平滑
- 质量判断：绿/黄/红三色帧质量分类

## 适用场景

- 单人棒球挥棒视频
- 单人投球视频
- 人物较完整、主体较清晰、遮挡不严重的视频
- 需要 2D 骨架、角度趋势、可视化 overlay 的项目流程

## 不适用或需谨慎

- 多人同框且目标人物不明确
- 人物太小、严重遮挡、半身画面
- 需要真实球速、球旋转、球路位移、真实触球瞬间的分析
- 需要医学诊断或专业训练结论的场景

## 主要输出

- 稳定骨架视频
- 每帧 33 个关键点 CSV
- 每帧质量判断 CSV
- 左右肘、左右膝角度 CSV
- 稳定性统计 JSON
